## Arduino Matrix Snake

**This was made using:**
- Elegoo Uno R3
- MAX7219 Led Matrix
- Four push buttons

Written and compiled using the the [arduino package](https://www.archlinux.org/packages/community/x86_64/arduino) on the arch repository.
